👶 Family Care

Family Care is a multilingual family health management application designed to help families monitor pregnancy, child health, medications, vaccinations, growth, and menstrual cycles in one secure and easy-to-use platform.

The application works offline and is designed for users who need reliable access to important health information anytime and anywhere.

🌍 Supported Languages

- English
- العربية (Arabic)
- ትግሬ (Tigre)

Users can switch languages directly within the application.

✨ Features

🤰 Pregnancy Tracker

- Track pregnancy progress week by week
- Pregnancy timeline and milestones
- Pregnancy tips and reminders
- Estimated delivery date calculations

👶 Child Management

- Store and manage child profiles
- Track developmental milestones
- Organize family health records

💊 Medicine Manager

- Record medications
- Medication schedules
- Daily dose reminders
- Track treatment progress

💉 Vaccine Tracker

- Upcoming vaccine schedules
- Vaccination history
- Reminder notifications

📈 Growth Monitoring

- Record height and weight
- Growth tracking over time
- Visual growth records

🩸 Menstrual Cycle Tracker

- Cycle tracking
- Fertility awareness
- Period reminders
- Cycle history

🧮 Health Calculators

- Pregnancy-related calculations
- Health-related utility tools
- Quick reference calculations

🔔 Notifications

- Medicine reminders
- Vaccine reminders
- Pregnancy notifications
- Important health alerts

📱 Offline First

- Works without internet access
- Local data storage
- Fast performance
- Privacy-focused design

🔒 Privacy

Family Care stores user information locally on the device whenever possible. Users maintain control over their personal and family health information.

📥 Installation

1. Download the latest APK from the Releases section.
2. Open the APK file.
3. Allow installation from unknown sources if prompted.
4. Complete installation.
5. Launch Family Care.

📋 Requirements

- Android 7.0 or later
- Sufficient device storage
